# Banking Backend (Minimal)

This is a minimal Spring Boot backend example for a banking system.
Run with:

```
mvn spring-boot:run
```

Endpoints:
- GET /api/accounts
- GET /api/accounts/{id}
- POST /api/accounts
- PUT /api/accounts/{id}
- DELETE /api/accounts/{id}

Database: H2 in-memory (console at /h2-console)
